# @ali/yen-support

## Usage

```js
var support = require('@ali/yen-support')

if (support.retina) {
  // current device has retina screen
}
else if (support.highDensity) {
  // current device has high density screen
}
```

## Development

```js
$ tnpm start
$ open http://localhost:5000/tets/runner.html
```
